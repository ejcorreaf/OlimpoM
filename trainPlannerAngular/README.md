# TrainPlanner - Frontend Angular

Frontend de gestión de rutinas de entrenamiento construido con **Angular 20** y **Bootstrap**.


TrainPlanner es una plataforma que permite a entrenadores personales crear y asignar rutinas de ejercicio a sus clientes (trainees), mientras que los administradores gestionan usuarios, ejercicios y rutinas del sistema.


**Proyecto de Fin de Ciclo (TFC) - Desarrollo de Aplicaciones Web**
