# TrainPlanner - Backend Laravel

Un backend moderno desarrollado con **Laravel 12** para gestionar rutinas de entrenamiento personalizadas. Sistema completo de gestión de ejercicios, rutinas y usuarios con autenticación basada en tokens.

**Proyecto de Fin de Ciclo (TFC) - Desarrollo de Aplicaciones Web**
